OrderForms allows you to take payments online from your website or ads with professionally designed order forms.

== Installation ==
In your WordPress dashboard go to Plugins > Add New and search for “orderforms” and you will be able to quickly install the latest version of the plugin.

Goto Orderforms > Settings to add your domain prefix
Then proceed in creating as many forms as you have
