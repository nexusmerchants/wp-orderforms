OrderForms.com WordPress plugin.

== Description ==

OrderForms allows you to take payments online from your website or ads with professionally designed order forms.

== Installation ==
1. Unzip orderforms.zip and upload the folder 'orderforms' to the '/wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress

or:

In your WordPress dashboard go to 'Plugins > Add New' and search for "orderforms" to quickly install
the latest version of the plugin.

== Usage ==

1. Navigate to 'OrderForms > Forms' and link as many forms as you have on OrderForms.com
2. Navigate to 'OrderForms > Settings' to change the default slug/prefix
